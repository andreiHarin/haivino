## How to use this folder

- Treat it as a **read-only knowledge base** about the reference site.
- When building Haivino pages, open the matching `03-content-*` or `06-design-*` file alongside.
- Anything we learn later (e.g. from screenshots the user shares, or extra page visits) → add a new numbered file. Don't rewrite history.

## Limitations

- The site uses heavy CDN/bot protection — direct HTML/asset downloads were blocked from this environment. Captured content is the rendered text only.
- The Italian "Menu" page does not contain dish listings; it points to an external/embedded menu we could not access.
- No images, fonts, or CSS were downloaded. If the user wants pixel-level reference, ask them for screenshots or a saved-page bundle.
